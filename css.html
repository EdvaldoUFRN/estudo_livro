A propriedade color faz com que você modifique a cor do texto dentro do elemento. O css interpreta cada elemento do HTMl como se ele estivesse dentro de uma caixa, e quando usamos a propriedade background-color a gente está colocando um cor para esse fundo da caixa. ferramentas como photoshop ou GIMP podem disponibilizar o hex da cor que você selecionou.
o contraste influência na leitura de textos. se você colocar um texto claro em um fundo escuro você vai facilitar mais a leitura. Para criar o fallback, você pode especificar uma cor como um código hexadecimal, nome de cor ou valor RGB, seguido pela regra que especifica um valor RGBA. Se o navegador entender as cores RGBA, ele usará essa regra. Caso contrário, ele usará o valor RGB
p.one { background-color: rgb(0,0,0); opacity: 0.5;} p.two { background-color: rgb(0,0,0); background-color: rgba(0,0,0,0.5);}

também podemos usar a propriedade hsl hue saturation e lightness hsla é o alpha. body { background-color: #C8C8C8; background-color: hsl(0,0%,78%);} p { background-color: #ffffff; background-color: hsla(0,100%,100%,0.5);}

A ideia de fallback é tipo você coloca uma coisa antes dela, aí se a proxima n funcionar ela vai pegar a anterior.

Padding aumenta a lacuna entre a caixa e o texto interior próprio. margin aumenta a lacuna entre a caixa atual e as outras.
