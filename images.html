Imagens

Guardar as imagens em pastas diferentes e também caso eu quiser em um site maior eu posso guardar em subtópicos

se eu for usar uma imagem apenas como visual, então eu preciso usar o alt = ""

o src é onde ele vai pegar o arquivo, o alt é para search engines e para leitores de tela. e o title é para mostrar quando a pessoa colocar o mouse sobre.

Você pegar e colocar altura e largura numa foto, ajuda a fazer com que o site carregue o restante do conteúdo sem precisar esperar que a imagem seja carregada.

Antes era possível você alinhar uma imagem a partir do atributo align, mas agora é feito apenas pelo css. na horizontal, sendo left e right e para vertical sendo top, middle e bottom

Dicas para imagens, salvar no formato correto, salvar no tamanho da foto corretamente, para a página n esticar ela. maioria das telas mostram apenas páginas da web a 72pixels por polegada. portanto salvar em uma resolução mais alta que é essa seria inutil.

Imagem que tiver muitas cores diferentes, usar o JPEG Usar GIF ou PNG quando salvar imagens com poucas cores ou com areas largas com a mesma cor. Reduzir o tamanho de uma imagem pode proporcionar uma rapidez no carregamento. PNG salva transparente.

O html 5 agora possibilitou a gente colocar uma imagem e a sua legenda, pode ser possível colocar mais de uma imagem, porém tem que ter a mesma legenda.
