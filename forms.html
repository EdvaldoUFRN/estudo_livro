forms

basicamente a gente vai fazer o usuário preencher dados e depois enviar eles para outra página.

a tag form, sempre deve ter o atributo action e normalmente terá method e id.

Action é para onde o formulário será enviado, é a url para a página do servidor irá receber.

O metodo get vai adicionar os valores do formulário ao fim da url especificada. ele é ideal para pequenos formularios. como caixas de pesquisa. quando você está apenas recuperando dados do servidor web(não enviando informações que devem ser adicionadas ou excluídas de um banco de dados.)

O metodo post os valores são enviados no cabeçalho http, deve usar se os user carregam arquivos longos ou que contem dados confidenciais, adiciona informações ou exclui informações de um banco de dados. O atributo id é usado principalmente por scripts para verificar se você inseriu informações em campos que exigem valores.

Temos também a tag button que é possível usar para deixar que os usuários tenham um controle maior ssobre os seus botões, sendo possível colocar elementos dentro do botão. type="hidden" é usado para se ter um controle de formulário oculto. eles não são mostrados na página, eles permitem que os autores da página adicionem valores a formulários que os usuários não podem ver. ex. um autor da pagina web pode usar um campo oculto para inidicar em qual página o usuário estava quando enviou um formulário.

O atributo name indica o nome do controle de formulário que está sendo enviado ao servidor, juntamente com o valor que o usuário selecionou.
